# Velokraft Auto Parts — Website

Official informational website for Velokraft Auto Parts, Estonia.

## Project Structure

```
velokraft-project/
│
├── index.html              → Main website file (open this in browser)
│
├── assets/
│   ├── css/
│   │   └── style.css       → All styling (colors, fonts, layout)
│   ├── js/
│   │   └── main.js         → All interactions (form, menu)
│   └── images/             → Add company images/logo here
│
└── README.md               → This file
```

## How to Edit

| What to change | Which file to open |
|---|---|
| Text, content, sections | index.html |
| Colors, fonts, spacing | assets/css/style.css |
| Form, menu behavior | assets/js/main.js |
| Images / Logo | assets/images/ folder |

## How to Deploy

### Netlify (Recommended)
1. Go to netlify.com
2. Drag and drop the entire `velokraft-project` folder
3. Get a live link instantly

### Railway
1. Go to railway.app
2. Upload the project folder
3. Set root to index.html
